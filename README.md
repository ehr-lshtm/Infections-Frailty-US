This repository contains analytical code for study titled : **Frailty progression following severe infections in adults ≥65 years in US and England: two matched cohort studies.** Only analytical code for the US component can be found in this repository. The code for the England component can be found in another repository Infections-Frailty-England.

Codelist used to defining variables are published here [codelists](https://doi.org/10.17037/DATA.00004966).
